# Requirements Elicitation

## Stakeholders
| Stakeholder | Description |
|------------|------------|
| Participant | NDIS client using services |
| Staff | Support workers managing shifts |
| Admin | Manages system operations |

## Identified Problems
- Manual data handling (paper/spreadsheets)
- No centralised system
- Delays in booking and communication
- Risk of data inconsistency

## Key Needs
- Online service booking system
- Staff scheduling and management
- Secure document storage
- Centralised platform for all users
