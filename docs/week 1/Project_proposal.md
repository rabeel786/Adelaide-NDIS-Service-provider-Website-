# Project Proposal

## Project Title

## Description
A web-based system to manage participants, staff scheduling, bookings, and documents.

## Problem Statement
Current system is manual, causing inefficiencies and data inconsistency.

## Objectives
- Digitise operations
- Improve efficiency
- Provide better user experience
